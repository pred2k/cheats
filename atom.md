
## Atom Shortcuts

key combination          | description
-------------------------|------------
`CTRL` + `SHIFT` + `L`   | Toggle Markdown
`CTRL` + `M`             | Go to matching Bracket
`⌘` + `SHIFT` + `L`      | split selections into lines
`⌘` + `K` then `⌘` + `U` | UPPERCASE
`⌘` + `K` then `⌘` + `L` | lowercase
